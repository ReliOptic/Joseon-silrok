**답: 라이트와 다크의 의도적 교차**

추가 요청: **교차에 의미적 근거가 있어야 합니다.**

현재는 L3 라이트 → L3.5 다크 → L4 라이트로 전환되는데 시각적으로만 단절이지 의미가 없음. 디자이너가 이 교차를 다음과 같이 정당화해주세요:

- 예: "공식 기록(라이트) ↔ 야사·환영(다크)"
- 예: "낮의 조정(라이트) ↔ 사관의 밤 기록(다크)"

이런 식의 의미 축이 잡혀야 색온도 교차가 단순 변주가 아닌 **연출**이 됩니다.

## Q9. 전환 드라마 강도

**답: 왕위 교체 방식별로 다르게**

구체:
- **정상 승계** (예: 세종→문종) → 절제, 1초 페이드 + 한 줄
- **반정/쿠데타** (예: 연산군→중종, 광해군→인조) → 강함, 3-4초 시네마틱 컷
- **단명/요절** (예: 인종 8개월) → 짧고 비통, 1초 안에 끝
- **양위/평양양위** (예: 정종) → 차분한 1.5초
- **왕위 찬탈** (예: 단종→세조) → 어둡고 긴 3-4초

이게 들어가면 27왕이 따로 노는 문제가 동시에 풀립니다. 데이터 단에 `succession` 메타데이터를 신설해 드릴 수 있습니다 (의뢰자 측 작업).

## Q12. 꼭 담고 싶은 디테일 / 피해야 할 것

### 꼭 담고 싶은 것

1. **모순의 시각화** — 영조처럼 위대한 군주이자 비극의 아버지인 인물의 모순을 한 화면 안에 담는 디자인 언어
2. **사신(史官)의 1인칭 보이스 살리기** — 데이터에 이미 있는 `commentary` 필드. "사신은 논한다"라는 1인칭이 디자인적으로 살아나야 함
3. **27왕 사이 위계 차별화** — 재위 기간, 역사적 무게의 차이가 L1에서 시각으로 느껴져야 함 (모든 왕 동등 사이즈 카드 X)
4. **모바일 한 손 엄지 도달** — 320 px에서도 한 손으로 끝까지 가능해야 함
5. **한자가 의미 단위** — 단순 장식 워터마크가 아니라, 그 사건의 핵심 한자가 의미를 갖고 작동하게
6. **한국어 타이포의 정확한 위계** — 한국어는 영어보다 위계 만들기가 어렵다는 점을 디자이너가 알고 풀어주길

### 피해야 할 것

1. **사극 OST 톤의 신비주의** — "한국적 신비"라는 외국인 시각의 클리셰
2. **왕정 미화** — 영광이든 비극이든 한쪽으로 치우치지 않게. 모순을 모순 그대로
3. **일본 NHK식 다큐 톤이 너무 강해지지 않게** — 한국사임을 절대 잊지 말 것 (NHK는 톤 레퍼런스일 뿐, 베끼는 게 아님)
4. **정치 색채 탈락** — 어느 시대든 정치적 해석이 가능. 특정 정치 진영과 결부될 수 있는 디자인 표현 금지 (예: 색상으로 보수/진보 연상 등)
5. **디지털 트렌드 클리셰** — 글래스모피즘은 이미 쓰고 있지만 그게 트렌드 추종으로 보이지 않게. 우리 톤에 정당화되어야 함
6. **일본 미감의 무의식 차용** — 미니멀 디자인이 일본 와비사비로 흐르는 흔한 함정 회피. 조선 미감은 일본 미감과 다릅니다
